Provides an easy way to list a few of your recent activities in a widget alongside your blog content.

== Description ==
A simple widget that displays recent activities from Runkeeper. It's configurable through Appearance --> Widgets. Configuration options are; user name, activity display count.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `runkeeper-widget.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance --> Widgets, find the Runkeeper Widget. Drag widget to right side and configure it.

== Frequently Asked Questions ==

= Do I need to enter my Runkeeper password? =

No. This widget just needs your Runkeeper user name. It pulls the data via a screen scrape of your public Runkeeper page.
